Vad är ett ram minne?

Ram minnet (random access memory) sparrar på olika sorters information så
att moderkortet kan ta den informationen snabbt och ge den till dig som person eller till någon komponent att göra något. Skälet till att man har ram minne är för att det går mycket snabbare än om man tar det från tex ett hdd eller ssd (hårddisk).

vad sparas på ett ram minne?

Ram minnet är som man kan kalla ett korttidsminne, för att det sparar en viss information en viss tid så att man kan gå tillbaka till det igen mycket snabbare än från hårddisken. Men efter ett tag så tar den bort det och lagrar ny information när du stänger av datorn.

Vad ska man tänka på när man ska köpa ram minne?

Man ska tänka på att köpa samma sorts ram minne för att om man köper
så kommer det inte att fungera med varandra.
